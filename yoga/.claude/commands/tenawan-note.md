# テナワン note記事作成

## 手順

### 1. 入力確認
ユーザーに以下を確認:
- キーワード（SEO対象）
- 読者が知りたいこと（検索意図）
- 記事タイプ（pricing/howto-dev/compare/service/generic）

### 2. ナレッジベース読み込み
`knowledge/tenawan/` 内の全ファイルを読み込み、石田社長の文体を学習

### 3. 調査
- Web検索で直近6ヶ月〜1年の情報を収集
- ファクトチェック実施
- 出典を記録

### 4. 構成案作成
- タイトル案（疑問形推奨）
- H2/H3の見出し構成
- 各セクションの要点

### 5. ユーザー確認
構成案を提示し、承認を得る

### 6. 執筆
CLAUDE.mdのライティングルールに従い執筆:
- 1文ごとに段落分け
- 1文80〜100文字
- 絵文字なし
- 石田社長のトーンを再現

### 7. 出力
Markdown形式で出力
