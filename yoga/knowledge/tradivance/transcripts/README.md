# 音声書き起こしナレッジベース

音声配信からの書き起こしを蓄積するフォルダです。

## 書き起こしプロセス

### 1. 音声ファイルの準備
- 対応形式: mp3, m4a, wav, mp4
- ファイル名: `YYYY-MM-DD_タイトル.拡張子`

### 2. 書き起こし実行
```
# Whisper API等で書き起こし
# 出力: transcripts/YYYY-MM-DD_タイトル_raw.md
```

### 3. 整形・編集
- 話者の区別（必要に応じて）
- 段落分け
- 誤字脱字の修正
- 出力: `transcripts/YYYY-MM-DD_タイトル.md`

### 4. ナレッジ抽出
重要な知見は別途`knowledge/tradivance/`直下にテーマ別ファイルとして整理

## ファイル命名規則

| 種類 | 形式 | 例 |
|------|------|-----|
| 生書き起こし | `YYYY-MM-DD_タイトル_raw.md` | `2026-02-12_AI活用術_raw.md` |
| 整形済み | `YYYY-MM-DD_タイトル.md` | `2026-02-12_AI活用術.md` |
