# LP制作テンプレート集

LP（ランディングページ）制作の練習・テンプレートをまとめたモノリポ。

## プロジェクト一覧

| フォルダ | ジャンル | 技術スタック |
|---------|---------|------------|
| `saas/` | SaaSサービス | Vite + Tailwind CSS |
| `cooking/` | 料理教室 | Vite + shadcn/ui |
| `yoga/` | ヨガスタジオ | Next.js |
| `postpartum-diet/` | 産後ダイエット | Vite + Tailwind CSS |
| `birukatsu/` | ビルカツ（不動産） | HTML単体 |
| `ai-sinc/` | AI Sinc Studio | Vite + Tailwind CSS |

## クライアント向けLP制作時

1. 近いジャンルのフォルダをコピー
2. 新リポ `client-{社名}-lp` を作成
3. カスタマイズして納品
